How to enjoy 3D Path Finder !
By dragging, you can relocate the start node and end node.
You can create random walls by clicking "GENERATE WALL(S)" button. Also, you can click the empty spots on the board to create walls
You can select a type of algorithms (BFS/DFS).
The speed options of visuzlizing path finder are fast(0.03s), normal(0.2s), and slow(0.4s).
Click "RESET" button to restart the entire board.
